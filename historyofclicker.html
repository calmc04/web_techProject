History of Clicker Games

Clicker games which are sometime referred to as incremental games are a type of game where the player completes a simple objective and in return gets a reward, this reward usually allows the player to "increase" their ability to get more rewards or make the action easier to achieve. The first clicker game is widely regarded as a game called "Progress Quest" this game featured lots of common features which are still seen today in these types of games, the most popular clicker game is Cookie Clicker. Other notable games include Clicker Heroes which has even been released as a whole game.
